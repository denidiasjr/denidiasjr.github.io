Patrimônios App

O aplicativo de Patrimônios foi desenvolvido para suprir a necessidade da Direção Administrativa do Campus Três Lagoas de obter informações dos patrimônios da instituição de forma mais prática, uma vez que nem sempre havia um computador próximo ao patrimônio de consulta. Sendo assim, o aplicativo foi desenvolvido para capturar as informações do Sistema Unificado de Administração Pública e apresentar para o usuário na tela do seu dispositivo Android.

Tecnologias que envolveram o projeto

  • Java (Gradle, Android Volley)
  • SQLite
  • PHP (WebService, cURL)
  • Android Studio
