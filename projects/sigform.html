SIGFORM - Sistema de Geração de Formulários

O Sistema de Geração de Formulários foi desenvolvido com o objetivo facilitar o preenchimento dos formulários utilizados dentro do IFMS. O projeto deu início em 2016 e foi finalizado em 2017. O sistema acabou não indo para produção, devido a atualizações feitas no SUAP (Sistema unificado de administração pública) que acabou por abranger o uso de boa parte do Sigform. Entretanto, o sistema ainda pode ser utilizado internamente no IFMS.

Tecnologias que envolveram o projeto

  • PHP (CakePHP, Composer, PDF-tk)
  • HTML + CSS (Bootstrap)
  • Javascript (JQuery)
  • Linux (Cron)
  • Vagrant
