Página de Ocorrências do IFMS

A página de ocorrências do IFMS foi criada com o intuito de estreitar a comunicação entre o setor de tecnologia da informação e os estudantes da instituição. Sendo assim, através da página de ocorrências o estudante poderá enviar relatar qualquer problema relacionado a TI que, no mesmo instante da abertura da ocorrência, a mesma é encaminhada para a caixa de email do setor. Com o mapeamento de IP realizando no Back End da página, é possível identificar de qual computador surgiu a ocorrência, tanto para agilizar o atendimento, quanto para fins de auditoria.

Tecnologias que envolveram o projeto

  • HTML + CSS
  • Javascript (JQuery, Ajax)
  • PHP (PHPMailer)
